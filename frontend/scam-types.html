대표적인 전세사기 유형

🏚️

깡통전세

고위험 전세가율
매매가와 전세가가 비슷해 집값 하락·경매 시 보증금 회수가 어려운 유형.
  • 전세가율 70% 이하 권장
  • 실거래가·시세 반드시 비교
  • 담보권(근저당) 총액 확인

사례: 매매 3.2억, 전세 3.0억 → 집값 하락 시 보증금 미회수 위험.

예방: HUG 전세보증보험 가입 가능 여부 확인, 보증금 분할지급 금지.

🏦

신탁사기

주의 신탁원부
신탁회사 동의 없이 임대계약을 체결해 무효가 되는 유형.
  • 등기부 '을구'에 신탁 여부 확인
  • 신탁원부 발급 및 임대 동의서 확인
  • 임대인과 신탁사 간 위임관계 서류 확보

사례: 신탁 부동산임에도 집주인이 단독으로 계약 체결.

예방: 신탁사 확인 전화, 임대동의서 원본 확인 후 계약.

📄

이중계약

고위험 신원확인
임차인이 집주인 행세를 하거나, 동일 주택에 중복 임대하는 유형.
  • 소유자 신분증·인감증명서 대조
  • 대리계약 시 위임장·인감증명서 확인
  • 계약금 계좌가 소유자 명의인지 확인

사례: 세입자 A가 집주인 행세로 B에게 전세 계약 후 잠적.

예방: 등기부 소유자 계좌로 송금, 현장 입회·영상기록 보관.

🚫

무등록 중개

주의 자격확인
중개업 등록 없이 불법으로 중개하여 계약서 위조·허위 정보를 제공하는 유형.
  • 중개사무소 등록증 확인
  • 공인중개사 자격증 원본 대조
  • 계약서에 중개업소 도장·서명 필수

사례: 미등록 브로커가 가짜 등기부로 계약 유도 후 사라짐.

예방: 지자체 홈페이지에서 중개업소 등록 여부 확인, 계약서 사본 보관.

🏢

빌라왕 사기

고위험 다세대
대출로 다세대 주택을 매입 후 고액 전세를 받아 빚만 남기고 파산하는 유형.
  • 건물 전체 임대 현황·선순위 확인
  • 소유자의 다른 부동산 등기 조회
  • 담보 대출 총액이 건물 시세 대비 과다한지 확인

사례: 빌라 여러 채를 대출로 매입 → 전세 받고 파산 신청 → 세입자 보증금 손실.

예방: 선순위 확정일자 순위 확인, HUG 보증 가입 여부 점검.
